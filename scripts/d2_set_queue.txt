d2_set_queue.dms  by rentan


Dorothy2/common/additem.dms で Mode=OptionQueue を使う時のための、
支援スクリプトです。

全てのキューフォルダのうち、Dorothy2 用の設定をしているフォルダの
フォルダ設定 → HTTP(1) → HTTP メソッド → POST する文字列に
	queue=/キューフォルダ名
という形式の文字列を追加します。

Shift キーを押しながら実行した場合は、追加した文字列を削除します。


インストール方法
================

1. d2_set_queue.dms を Irvine の scripts フォルダにコピーします。

2. ツール → オプション設定を開き、スクリプトのタブを開きます。

3. 「更新」ボタンをクリックします。

4. OnMainMenuClick をクリックします。

5. 「キューフォルダ名をオプション設定」にチェックを入れます。

6. OK ボタンで閉じます。


キューフォルダ名を設定する方法
==============================

1. ツール → メインメニュー → 「キューフォルダ名をオプション設定」を
   クリックします。

2. 実行後に結果が表示されます。左端の文字の意味は以下のとおりです。
	無視 → Dorothy2 用のキューフォルダではないため無視しました。
	設定 → 設定しました(成功)。
	失敗 → 既に入力されている文字列がオプション形式ではないため
	        設定できませんでした。


キューフォルダ名を削除する方法
==============================

1. ツール → メインメニュー → 「キューフォルダ名をオプション設定」を
   Shift キーを押しながらクリックします。

2. 実行後に結果が表示されます。左端の文字の意味は以下のとおりです。
	無視 → Dorothy2 用のキューフォルダではないため無視しました。
	削除 → 削除しました(成功)。
	失敗 → 入力されている文字列がオプション形式ではないため
	        削除できませんでした。


アンインストール方法
====================

1. ツール → オプション設定を開き、スクリプトのタブを開きます。

2. 「キューフォルダ名をオプション設定」のチェックを外します。

3. OK ボタンで閉じます。

4. Irvine の scripts フォルダにある d2_set_queue.dms を削除します。


仕様・制限事項
==============

・Dorothy2 用の設定がされているフォルダと見なす条件
	1. HTTP メソッドが GET になっている。
	2. スクリプトの「動画ダウンロード２Ａ」にチェックが入っている。

・文字列は Dorothy2/common/options.dms に対応した形式です。既に他の設定が
  入力されている場合は
	profile=0;queue=/キューフォルダ名
  のように ; で連結されます。
