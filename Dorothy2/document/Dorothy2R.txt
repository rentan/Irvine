Dorothy2R by rentan


Dorothy2R is modified version without permission of xor's Dorothy2.
This is not intended for general use users.

Behavior is incompatible with Dorothy2.
Site script works almost.

Do not request support about Dorothy2R to xor author of Dorothy2.


Environment
===========

Windows Vista/7/8/8.1
Irvine 1.3.1
Dorothy2 latest version


Usage
=====

1. Install Dorothy2 to perfection.

2. Copy files to Dorothy2 folder.

3. Check scripts at Irvine option setting dialog.

4. Uncheck Dorothy2A.dms at folder setting dialog.

5. Check scripts at folder setting dialog.

6. Save setting by "Doroty2R setting".
   (This creates scripts\Dorothy2R_ini.dms)


Caution with Dorothy2
=====================

- Do not check both Dorothy2A.dms and Dorothy2R_a.dms at same folder.

- Save setting even by "douga donwload2 setting" not only
  "Dorothy2R setting".


Limitations
===========

- Use Dorothy2B.dms if needed.

- Since not load xor.dms, scripts that rely on xor.dms does not work.

- changeFileName() returns argument. Use NameBuilder#normalize() and/or
  Utf8#decodeCharRef() instead of it.
