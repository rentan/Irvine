common\getterobject.dms


youtube_info.dms などから使用しているオブジェクトです。


使用方法
========

  common_load ('getterobject');

  // constructor
  void function MyObject () {
  };

  GetterObject.expand (MyObject);

  var myobj = new MyObject ();


GetterObject オブジェクトのプロパティ
=====================================

expand (obj)
  obj ... object: 拡張するオブジェクト

  obj を拡張します。
