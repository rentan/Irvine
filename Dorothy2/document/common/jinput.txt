Usage
=====

  common_load ('jinput');
  var jinput = new JInput ();
  var text = jinput.inputText ();


Jinput object's properties
==========================

text ... string


JInput object's methods
=======================

constructor
new JInput ()
  return value ... JInput instance

  e.g.  var jinput = new JInput ();

inputText (hta_opt)
  hta_opt ... array or object: dialog option
  return value ... string or null

  hta_opt = {
    title: 'dialog title',
    width: dialog_width,
    height: dialog_height,

    limit: seconds to abort automatically,
    extend: seconds to abort after input,

    min: min text length (default: 1)
    max: max text length (default: 255)

    image: 'image-data',  // binary data as string

    p: 'text to print'
  };

synchronize (func)
  func ... function: callback function (no arguments, no return value)
