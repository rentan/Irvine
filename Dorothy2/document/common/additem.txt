AddItem is a object for adding queue items to Irvine.

See http://rentan.org/blog/tag/additem/ for details.


Usage (for users)
=================

[AddItem]
Mode=
Queue=
Listname=


Mode ... Auto, AddQueueItem, Import or OptionQueue

- Mode=Auto
  Same as AddQueueItem.

- Mode=AddQueueItem
  Repeat AddQueueItem.

  Queue folder is specified by Queue=.

- Mode=Import
  Save items as list file and Import.

  Filename of list file is specified by Listname=.

- Mode=OptionQueue
  repeat AddQueueItem.

  Queue folder is specified by queue= in option setting.


Usage (for developers)
======================

  common_load ('additem');

  var ini = { Mode: 'Auto', Queue: '/Default', Listname: '' };
  var additem = new AddItem (ini);

  var list = [
    { Url: 'http://example.com/', Filename: 'index.html' }
  ];
  additem.send (list) || exit ();


AddItem object's properties
===========================

mode ... string
  'Auto',  'AddQueueItem' or 'Import'

queue ... string

listname ... string

iniTemplate ... object


AddItem object's methods
========================

constructor
new AddItem (opt)
  opt ... object: option (omittable)
  return value ... AddItem instance

  e.g.  var additem = new AddItem ({ Mode: 'Import' });

send (list)
  list ... object or array: list to download
  return value ... boolean
