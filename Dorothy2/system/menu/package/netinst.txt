netinst.dms -- ネットインストール for Dorothy2


https://github.com/rentan/Irvine から ZIP ファイルをダウンロードして
各サイト用のスクリプトをインストールするツールです。

無保証です。

使用する場合は、必ずあらかじめ全ドライブのバックアップを取って下さい。


動作環境
========

Windows Vista/7/8/8.1
Irvine 1.3.1
Dorothy2 最新版


使用方法
========

1. Dorothy2 の最新版が完全に動作する環境を構築する。
   解凍コマンドも設定する。

2. 全ドライブのバックアップを取る。

3. netinst.dms を以下のフォルダにコピーする。
   Dorothy2\system\menu\package\

4. netinst.txt の文字コードを Shift_JIS に変換した上で、以下のフォルダ
   にコピーする。
   Dorothy2\document\メニュー\パッケージ\

5. 「動画ダウンロード２設定」を開く。

6. メニュー → パッケージ → ネットインストールをクリックする。

ネットインストールのウィンドウが開く。

7. 「ダウンロード」ボタンをクリックする。

8. インストールしたいスクリプトにチェックを入れる。

9. 「インストール」ボタンをクリックする。

10. 終了ボタンをクリックして終了し、「動画ダウンロード２設定」
    でデータを再読み込みする。


詳細表示(ログ)の説明
====================

copy
  新規ファイルをコピーした。

update
  既存ファイルに上書きした。

same
  同一内容のファイルが存在した(コピー省略)。

skip
  ファイル内容は異なるが、バージョン番号が同一なのでコピーしなかった。
  program\*.dms のみ適用。

error
  コピー時にエラーが発生した。


仕様・制限事項
==============

・「新しいバージョンをインストールする」という動作ではなく
  「チェックを入れたスクリプトをインストールする」という動作です。

  同じバージョンや、より新しいバージョンがインストール済みでも
  チェックを入れたスクリプトは必ずインストールが行われます。

  ただし内容が同じファイルであれば上書きはしません。

・program\*.dms については内容が異なっていても、バージョン番号が同じ
  なら同一とみなして上書きをしません(ログは skip 表示)。

・終了時に、Dorothy.temp フォルダに作業用フォルダ/ファイルを残した
  ままにします。
